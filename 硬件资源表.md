# 硬件资源表 - ymq_2

## 基本信息

| 项目 | 内容 |
|------|------|
| 芯片型号 | STM32H750VBT6 (STM32H750VBTx) |
| 封装 | LQFP100 |
| 开发框架 | STM32 HAL 库 (STM32Cube FW_H7 V1.12.1) |
| IDE | Keil MDK-ARM V5.32 |
| 代码生成 | STM32CubeMX 6.14.1 |

## 时钟配置

| 时钟 | 频率 |
|------|------|
| CPU (Cortex-M7) | 480 MHz |
| SYSCLK | 480 MHz |
| HCLK (AHB) | 240 MHz |
| APB1 | 120 MHz |
| APB2 | 120 MHz |
| APB3 | 120 MHz |
| APB4 | 120 MHz |
| TIM2 时钟 (APB1×2) | 240 MHz |
| 振荡器来源 | HSI (内部) → PLL (PLLM=4, PLLN=60, PLLP=2) |

> 注意：.ioc 中配置了 PH0/PH1 为 HSE 引脚，但 main.c 中 SystemClock_Config 使用 HSI 作为 PLL 源

## 定时器配置

| 定时器 | 模式 | 周期 | 用途 |
|--------|------|------|------|
| TIM2 | PWM 输出 (4通道) | 11999 (ARR) | 4路 PWM 电机驱动，频率 = 240MHz/(0+1)/(11999+1) = 20kHz |
| TIM3 | 编码器模式 TI12 | 65535 | 电机A编码器 (PA6/PA7) |
| TIM4 | 编码器模式 TI12 | 65535 | 电机B编码器 (PD12/PD13) |
| TIM5 | 编码器模式 TI12 | 4294967295 (32位) | 电机C编码器 (PA0/PA1) |
| TIM8 | 编码器模式 TI12 | 65535 | 电机D编码器 (PC6/PC7) |
